
Dark Matrix Tic-Tac-Toe Game 

This is a simple Tic Tac Toe game I built using Python and Tkinter(for user interface)
I gave it a dark “Matrix-style” theme just to make it look more modern and fun instead of a plain basic game.

 Working of the game
 
Two players can play (X vs O)
Turns switch automatically after each move
The game instantly detects when someone wins
Winning line gets highlighted so it’s easy to see
Keeps track of scores for X, O, and draws
You can restart the game anytime with one button
It won’t allow invalid moves (like clicking the same box twice)

Purpose  of developing this game?

I made this project to practice Python and GUI development.
It’s a small beginner-friendly game, but I tried to make it look more stylish with a dark “Matrix” theme and smoother gameplay experience.
